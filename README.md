# 🛒 買い物メモアプリ

普段の生活で気づいた「買わなきゃ」をメモしておき、店タイプ（スーパー／ドラッグストア／百均／モール系）と売り場カテゴリ別に自動整理してくれるアプリ。URLを共有するだけで夫婦間でリストを共有できます。

- フロントエンド: `index.html`（単一ファイル、スマホ優先）
- サーバー: `app.py`（Flask）
- AI: Claude Haiku で商品名から店タイプ・売り場カテゴリを自動推定（手動修正も可）
- 共有: ログイン不要。リストごとのURL（`/l/<リストID>`）を知っている人だけがアクセス可能
- データ: `lists_data.json` にリストIDごとの買い物リスト・よく買うものリストを保存

## ローカルで起動

```
start.bat をダブルクリック
→ http://localhost:5053 にアクセス（自動でリストURLにリダイレクトされます）
```

初回のみ `shopping_config.json` に `anthropic_api_key` を設定してください（Git管理外）。

## Render にデプロイ

1. このフォルダを GitHub リポジトリに push
2. Render ダッシュボード → **New → Blueprint** → リポジトリを選択（`render.yaml` を自動検出）
3. デプロイ後、**Environment** タブで `ANTHROPIC_API_KEY` を設定

## 使い方

1. 発行されたURLをパートナーに共有（LINEなど）
2. お互いに商品名を入力すると、AIが店タイプ・売り場カテゴリを自動で振り分け
3. カテゴリが違う場合はタップして手動修正
4. 買ったらチェックで即リストから削除。「よく買うもの」に自動登録され、次回はワンタップで再追加可能
